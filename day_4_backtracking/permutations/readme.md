# Permutations of Distinct Integers

## Problem Statement

Given an array `nums` of distinct integers, return all the possible permutations of the array elements. You can return the answer in any order.

### Input

- An array `nums` containing **distinct integers**. 

### Output

- An array containing all possible permutations of the given array `nums`. Each permutation should be an arrangement of the arrayâ€™s elements in a different order.

---

## **Example 1:**

**Input:**
```javascript
nums = [1, 4]
```

**Output**
```javascript
[[1, 4], [4, 1]]
```

## **Example 2:**

**Input:**
```javascript
nums = [1, 4, 5]
```

**Output**
```javascript
[[1, 4, 5], [1, 5, 4], [4, 1, 5], [4, 5, 1], [5, 1, 4], [5, 4, 1]]
```
